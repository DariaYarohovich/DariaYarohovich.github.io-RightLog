Инвентаризация товаров в магазинах

RightLog оптимальное решение для автоматизации процесса инвентаризации.

Решение основано на мобильной платформе RightLog и совместимо c 1С.

Позволяет:

  • осуществить выгрузку базы Номенклатуры товара из учетной системы в ТСД;
  • количественный сбор информации;
  • контроль соответствия товара наименованию из базы Номенклатур;
  • после завершения сбора информации данные выгружаются в учетную систему пользователя.

Дополнительно:

  • работа в режиме Online (не требует загрузки базы Номенклатуры в ТСД, поиск товара по ШК осуществляется напрямую из учетной системы через WiFi);
  • мониторинг актуальной цены товара из базы данных.
